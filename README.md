<h1 align="center"> DS-Spider </h1>

This project is a simple script for .DS_Store spidering. It downloads and parse .DS_Store files recusively. 


<br>

### Dependencies
This project can used with Python 2 and 3. Some external libs are requered.
```bash
pip install ds-store requests
```

<br>

### How to use
```bash
python ds-spider.py <url>
```
